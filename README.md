#Cocktail Library (iOS)

Generate and recommend cocktail recipes based on user selections and preferences. Users choose a base spirit, style (sour, spritz, frozen, etc.), and flavor/strength sliders; the app finds the closest classic recipes—and can synthesize a balanced AI-crafted recipe when nothing matches closely.

##Architecture
iOS App (SwiftUI)
SWift Package Manager 
Data(JSON) - 100 cocktails recipe
write login in VS Code
Simulate/Preview with Xcode
